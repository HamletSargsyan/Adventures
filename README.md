## Зависимости


* python (3.8+)
* pip
* curl (Нужен для установки)


## Установка и запуск

```bash
curl https://raw.githubusercontent.com/HamletSargsyan/Adventures/main/install.sh | bash
```
Запуск
```
cd Adventures/Adventures
python main.py # или python3 main.py
```
