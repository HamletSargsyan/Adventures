# Adventurs

**Adventurs** - Это текстовая игра, написанная на языке Python. Игра позволяет игроку исследовать окружающий мир, сражаться с противниками и и т.п.


**Версия**: 1.0.0-BETA

## Зависимости

``` python
colorama==0.4.6
```

## Установка
``` shell
pip install -r requirements.txt
```
